# Rho-Calculus with Functions for Alpha Equivalence

## Functions: Proc -> Set
* `N`: Names
* `V`: Variables
* `Q`: Quoted processes
* `BN`: Bound names
* `FN`: Free names
* `BV`: Bound variables
* `FV`: Free variables
* `CH`: Channel names
* `RN`: Receive names
* `RV`: Receive variables
* `SN`: Send names
* `SV`: Send variables
* `CH`: Channels
* `PP`: Process patterns
* `NP`: Name patterns
